## Artefactos

| Artefacto |         Propósito               | Responsable |
|-----------|---------------------------------|-------------|
| Exercise  | Definir el requisito funcional  | Yo    |
| Explore   | Explorar alternativas y riesgos | IA          |
| Review    | Validar y ajustar decisiones    | Yo   |
| Proposal  | Formalizar el cambio            | IA          |
| Design    | Diseñar la solución técnica     | IA          |
| Tasks     | Planificar la implementación    | IA          |
| Apply     | Generar el código               | IA y Yo     |
| Retro     | Registrar aprendizajes          | Yo   |
